# Next + TypeScript + Yarn monorepo

This has:

* an application workspace
* a business core workspace
* a shared component library workspace


- `$ git clone git@github.com:delete/next-monorepo.git`
- `$ cd react-native-web-monorepo`
- `$ yarn`
- `$ yarn app:dev`
